# Cape Town Web GIS (Mini Project)

## Overview

This project is a mini Web GIS application developed as part of my ongoing self-directed learning in geospatial technologies and web-based GIS. The application is built using Leaflet.js and is centred on the Cape Town metropolitan area.

The purpose of the project is to translate core Geomatics and GIS concepts into a functional, interactive web mapping environment, while strengthening my coding and problem-solving skills.

---

## Features

* Interactive map centred on the Cape Town metro area
* Multiple basemaps (OpenStreetMap and satellite imagery)
* Vector data representation (points, buffers, and polygons)
* Layer control for toggling map content
* Live coordinate display
* Scale bar for distance awareness
* Click-to-identify functionality for spatial referencing
